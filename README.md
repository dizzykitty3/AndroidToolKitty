## Features

* **Auto-Clear Clipboard**
* **System Shortcuts**: Quick navigation to deep system settings (e.g., Developer Options, Install Unknown Apps, ...)
* **Social Finder**: Directly open social media profiles via username
* **Year Progress**
* **Dev References**: Android API versions, device info, connected devices, ...
* **Design References**: Material Design color palette, Compose typography, font weight, ...

## Tech Stack

Written in Kotlin with Jetpack Compose for UI.

Supports Material You dynamic theming, edge-to-edge display, and other modern Android standards.

Built with the help of these libraries:
1. Android Jetpack
2. AboutLibraries
3. LeakCanary
4. Timber

## Installation

1. Go to the [Releases](https://github.com/dizzykitty3/AndroidToolKitty/releases) page.
2. Download the latest `.apk` file.
3. Open the file on your Android device and follow the prompts.
